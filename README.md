# Product Sales Program

This project is a simple Python program to register product sales. The program asks the user for some information and calculates the total cost of the products.

## Description

This project is a basic sales program made with Python. It is a small system that helps the user register products and calculate the total price.

The program asks the user for the name of the product, the quantity, and the price. After that, the program calculates the total cost of the product.

The system stores the information in a list. While the program is running, it is saving the sales information.

This project is useful to practice Python functions, user input, and simple calculations.

## Getting Started

### Dependencies

To run this program you need:

- Python 3
- A code editor like Visual Studio Code
- A computer with Windows, Mac, or Linux

Example:
- Windows 10

### Installing

1. Download the project from GitHub.
2. Open the folder in your code editor.
3. Make sure Python is installed on your computer.

## Executing program

To run the program follow these steps:

1. Open the Python file.
2. Run the program.
3. Enter the product name.
4. Enter the quantity.
5. Enter the price.
6. The program calculates the total cost.
7. The program asks if you want to add another product.
